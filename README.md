# PinkWave-
Веб-орієнтована інформаційна система для підготовки та публікації телевізійних новин

📄 PinkWave News — Панель редактора (Дипломний проєкт)
Опис проєкту
PinkWave News — це веб-додаток для керування новинами, розроблений як дипломний проєкт. Він включає:
Авторизацію редактора
Додавання, редагування та видалення новин
Завантаження медіа (зображення, відео, GIF)
Перегляд списку всіх опублікованих новин
Проєкт реалізовано з використанням HTML, CSS та JavaScript та зберігає дані у Firebase Firestore/Storage.
⚠️ Важлива особливість: наразі редактор жорстко прописаний у JS і тільки один. Додаткової реєстрації та авторизації немає, оскільки для дипломного проєкту це ускладнює задачу і немає потреби підключати повноцінну базу користувачів.
Структура проєкту
PinkWave-News/
│
├─ Autorization_blank/
│   ├─ authorization.html     # Сторінка логіну редактора
│   ├─ authorization.js       # Скрипт авторизації з жорстко заданим користувачем
│   └─ authorization.css      # Стилі сторінки логіну
│
├─ Editor_blank/
│   ├─ editor.html            # Панель редактора
│   ├─ editor.js              # Скрипт роботи редактора (CRUD)
│   └─ editor.css             # Стилі панелі редактора
│
├─ img/
│   └─ favicon.png            # Іконка сайту
│
└─ index.html                 # Головна сторінка (за потреби)
Технології
HTML5 / CSS3
JavaScript (ES6 Modules)
Firebase Firestore (для збереження новин)
Firebase Storage (для збереження медіа)
💡 Вибір Firebase обґрунтований спрощенням задачі для дипломного проєкту — немає потреби розгортати власну базу даних, авторизація мінімальна і достатня для демонстрації.
Авторизація
Тільки один редактор, дані якого прописані у JS-файлі authorization.js:
const VALID_EMAIL = "yuliia.horbatiukova.editor@pinkwave.ie";
const VALID_PASSWORD = "12345yullia";
Авторизація перевіряє email і пароль та зберігає isEditorLoggedIn у localStorage.
Якщо користувач не авторизований, доступ до панелі редактора блокується і відбувається редирект на authorization.html.
CRUD новин
Додавання
Поля форми:
Заголовок
Категорія
Текст новини
Медіа (опційно)
Медіа можна завантажити через drag-and-drop або кнопку вибору файлу.
Файл завантажується в Firebase Storage, посилання зберігається в Firestore.
Редагування
Кнопка Редагувати заповнює форму даними обраної новини.
Старе медіа видаляється при завантаженні нового файлу.
Після оновлення дані зберігаються у Firestore.
Видалення
Кнопка Видалити видаляє:
Файл із Firebase Storage (якщо є)
Документ новини з Firestore
Перед видаленням відображається підтверджуюче вікно.
Завантаження медіа
Підтримуються:
Зображення (image/*)
Відео (video/*)
GIF (image/gif)
Максимальний розмір: 20 МБ
Прев’ю файлу відображається перед завантаженням
Прогрес завантаження показується у вигляді смужки
Відображення новин
Новини відображаються у вигляді карток з:
Медіа (якщо є)
Заголовком та текстом
Категорією
Кнопками редагування та видалення
Сортування за датою створення — нові новини зверху.
UX та повідомлення
Повідомлення про успіх/помилку/інформацію показуються у верхній частині форми:
Зелений — успіх
Червоний — помилка
Рожевий — інформаційне повідомлення
Адаптивний дизайн під мобільні пристрої.
